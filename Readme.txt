本次项目基于C#窗体应用进行实现，共有两种运行方式：
-------------------------------------------------------
1.打开项目源代码中.sln文件，并利用Visual Stdio进行运行。
	项目使用Visual Stdio2022版（其他版本理论可行）
	需要在VS中安装C#窗体应用组件；
	可以在Debug或Release模式下运行；
-------------------------------------------------------
2.打开可执行文件文件夹，找到其中.exe文件；
	直接双击运行即可。
